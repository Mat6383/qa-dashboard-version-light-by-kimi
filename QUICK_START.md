# ⚡ Quick Start

> Voir le [README.md](README.md) complet pour la documentation détaillée.

## Prérequis
- Node.js ≥ 18
- npm ≥ 9

## Installation

```bash
npm install
```

## Démarrage

```bash
# Backend + Frontend en parallèle
npm run dev

# Backend seul
npm run dev -w backend

# Frontend seul
npm run dev -w frontend
```

## Tests

```bash
npm test
```

## Build production

```bash
npm run build
```
